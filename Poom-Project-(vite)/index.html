What is the role of the index.html file?

It's the main entry point of a website. It loads first when visiting a web page.

What is the URL you used to view the page? Is it HTTP or HTTPS?

Example: http://127.0.0.1:5500/index.html (HTTP) or https://example.com (HTTPS). The protocol is shown in the address bar.

How can you tell the page is being served locally vs from a real domain?

Local URLs use localhost or 127.0.0.1. Real domains look like example.com.

What tools did you use to inspect the response?

Browser Developer Tools, mainly the Network tab to inspect responses.
